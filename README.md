## 🙋🏻‍♂️External-Brain이 무엇인가요?
> External Brain은 `외부 뇌`라는 의미로, **저의 뇌역할을 대신하는 공간입니다.**<br>
<br>

저는 이것들 말고도 기억할 것들이 너무 많은 관계로 인터넷의 힘을 빌릴까 합니다.<br>
제 뇌가 기억하기엔 너무도 많은 양들의 정보들을 까먹지 않기 위해 기록하고, 정리하는 용도로 사용하고 있습니다.<br>

<br>
